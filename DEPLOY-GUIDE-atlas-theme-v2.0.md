# atlas-theme-v2.0 部署指南 —「墨綠電力藍圖」全站改版（含 v1.0 通行碼閘門）

⚠️ 部署到 **TW-US-power-atlas** repo（不是 gf-website）
⚠️ 本包**已內含 atlas-gate-v1.0 的通行碼閘門**，若尚未部署 v1.0，直接上傳本包即可一次完成。

## 設計概念
自主站的「航海圖」語彙獨立而出，改以**電網藍圖**為軸：
| 角色 | 主站（航海） | Atlas 新版（電力） |
|---|---|---|
| 底色 | 墨海藍 #0A1C30 | **墨綠 #0A2A21**（變電箱綠） |
| 紙色 | 海圖紙 #F4F6F7 | **電網圖紙 #F2F6F3**（帶綠調） |
| 主 accent | 航道青 #0FA3A8 | **電流翠青 #0E9E86** |
| 次 accent | 舵燈琥珀 #E8A23D | **高壓電力黃 #F0B429**（更亮、更「帶電」） |
| 中性灰 | 岸線灰（藍調） | **苔灰**（綠調，全套灰階同步換色） |
| Hero 格線 | 白色細格 | **翠青微光電網格線**（rgba 72,224,182） |
| Logo | 藍底青環黃電 | 墨綠底＋翠青虛線環＋電力黃閃電 |

**刻意保留不動**：所有圖表／地圖的「資料編碼色」——燃氣藍、燃煤灰、核能紫、綠電綠、水力青、太陽能黃、警示紅、電價漸層（綠→黃→紅）——確保數據判讀不受改版影響。

## 部署方式
GitHub 網頁 → `FalconKenny/TW-US-power-atlas` → Upload files → 拖入解壓後全部檔案與 `assets` 資料夾 → Commit 覆蓋 → 等 Pages 重建 1–2 分鐘。
（若瀏覽器看到舊配色，請強制重新整理 Ctrl+Shift+R 清快取）
